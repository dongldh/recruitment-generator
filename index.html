울산대학교 채용공고 생성기 ver1.0 (2026.03.23.)

제작: 울산대학교 총무처 총무인사팀

📋 기본 정보
👁️ 미리보기
정보를 입력하고 "미리보기 생성" 버튼을 클릭하세요.

📖 채용공고 생성기 사용 가이드

🚀 빠른 시작

필수 항목(*)만 입력하고 "미리보기 생성" 버튼을 클릭하면 채용공고가 자동으로 생성됩니다.

🤖 AI 자동 입력 (선택사항)

기존 채용공고 텍스트를 복사해서 붙여넣으면 AI가 자동으로 각 항목에 입력합니다.

  1. 기존 채용공고 텍스트 복사
  2. "AI 자동 입력" 박스에 붙여넣기
  3. "🤖 텍스트 자동 분석" 버튼 클릭
  4. 자동으로 입력된 내용 확인 및 수정
💡 AI 기능 사용 방법 AI 기능은 웹서버 환경에서만 작동합니다. 로컬 파일로 열면 "Failed to fetch" 에러가 발생합니다.

해결 방법:
1. 명령 프롬프트(CMD) 실행
2. cd 바탕화면 입력
3. python -m http.server 8000 입력
4. 브라우저에서 http://localhost:8000 접속

※ Python이 없으면 수동 입력을 사용하세요 (AI 없이도 모든 기능 정상 작동)

📝 필수 입력 항목

  • 공고명: 예) 인권센터 계약직
  • 모집 부서: 예) 인권센터
  • 고용 형태: 정규직, 기간제, 무기계약직, 계약직 중 선택
  • 모집 인원: 예) 1명
  • 근무 시작 예정일: 날짜 선택
  • 지원 마감일: 날짜 선택

✅ 미리보기 및 저장

  1. ✓ 미리보기 생성: 오른쪽에서 실시간 미리보기
  2. 📋 HTML 복사: HTML 코드를 클립보드에 복사 (에디터에 붙여넣기 가능)
  3. 🖨️ 인쇄: 바로 인쇄
  4. 💾 다운로드: HTML 파일로 저장
⚠️ 주의사항 다운로드 전에 반드시 미리보기를 확인하세요!

🆘 문제 해결

AI 기능 오류 시: 수동으로 입력해주세요.

미리보기가 안 보일 때: 필수 항목(*)이 모두 입력되었는지 확인하세요.

날짜 경고가 뜰 때: 마감일이 시작일보다 빠른지 확인하세요.
