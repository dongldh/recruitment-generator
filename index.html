🚀 빠른 시작

필수 항목(*)만 입력하고 "미리보기 생성" 버튼을 클릭하면 채용공고가 자동으로 생성됩니다.

📝 필수 입력 항목

  • 공고명: 예) 인권센터 계약직
  • 모집 부서: 예) 인권센터
  • 고용 형태: 정규직, 기간제, 무기계약직, 계약직 중 선택
  • 모집 인원: 예) 1명
  • 근무 시작 예정일: 날짜 선택
  • 지원 마감일: 날짜 선택

💡 마크다운 리스트 사용법

주요 업무, 지원 자격, 우대 사항 등을 입력할 때 다음과 같이 작성하면 자동으로 예쁜 리스트로 변환됩니다:

✨ 불릿 리스트 (-, *, •)

- 인권 상담 및 사건 처리
- 인권 교육 기획 및 실행
- 센터 운영 지원

✨ 번호 리스트 (1., 2., 3.)

1. 서류 전형
2. 면접 전형
3. 최종 합격 발표

✅ 미리보기 및 저장

  1. ✓ 미리보기 생성: 오른쪽에서 실시간 미리보기
  2. 📋 HTML 복사: HTML 코드를 클립보드에 복사
  3. 🖨️ 인쇄: 바로 인쇄
  4. 💾 다운로드: HTML 파일로 저장
⚠️ 주의사항 복사 또는 다운로드 전에 반드시 미리보기를 확인하세요!

🆘 문제 해결

미리보기가 안 보일 때: 필수 항목(*)이 모두 입력되었는지 확인하세요.

날짜 경고가 뜰 때: 마감일이 시작일보다 빠른지 확인하세요.
