# 울산대학교 채용공고 생성기 v1.0

제작: 울산대학교 총무처 총무인사팀
날짜: 2026.03.23

## 📋 파일 목록

1. **index.html** - 채용공고 생성기 메인 파일
2. **서버실행.bat** - 웹서버 실행 배치 파일 (AI 기능 사용 시 필요)
3. **README.txt** - 사용 설명서 (이 파일)

## 🚀 빠른 시작

### 방법 1: 바로 사용 (AI 기능 없이)
1. `index.html` 파일을 더블클릭
2. 필수 항목(*) 입력
3. "미리보기 생성" 버튼 클릭
4. 다운로드 또는 인쇄

### 방법 2: AI 기능 사용
1. `서버실행.bat` 파일을 더블클릭
2. 명령창이 열리면 브라우저에서 `http://localhost:8000` 접속
3. AI 자동 입력 기능 사용 가능
4. 종료: 명령창에서 Ctrl+C

※ Python이 설치되어 있어야 합니다.

## 📝 주요 기능

✅ 필수 항목 입력으로 자동 채용공고 생성
✅ AI 자동 입력 (텍스트 붙여넣기 → 자동 분석)
✅ 실시간 미리보기
✅ HTML 코드 복사
✅ 인쇄 기능
✅ HTML 파일 다운로드

## 💡 사용 팁

1. **AI 자동 입력**
   - 기존 채용공고 텍스트를 복사
   - "AI 자동 입력" 박스에 붙여넣기
   - "🤖 텍스트 자동 분석" 버튼 클릭
   - ※ 웹서버 실행 필요

2. **수동 입력**
   - AI 없이도 모든 기능 사용 가능
   - 필수 항목(*)만 입력하면 됨

3. **저장 방법**
   - 📋 HTML 복사: 코드를 클립보드에 복사
   - 💾 다운로드: HTML 파일로 저장
   - 🖨️ 인쇄: 바로 인쇄

## ⚠️ 문제 해결

### AI 기능 "Failed to fetch" 에러
→ `서버실행.bat` 실행 후 `http://localhost:8000` 접속

### Python이 없다고 나옴
→ AI 기능 사용 불가, 수동 입력 사용

### 미리보기가 안 보임
→ 필수 항목(*)을 모두 입력했는지 확인
