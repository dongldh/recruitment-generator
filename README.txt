# 울산대학교 채용공고 생성기 v1.0

제작: 울산대학교 총무처 총무인사팀
날짜: 2026.03.23

## 📋 파일 목록

1. **index.html** - 채용공고 생성기 메인 파일
2. **server.py** - AI 기능용 프록시 서버 (Python)
3. **서버실행.bat** - 서버 실행 배치 파일 (Windows)
4. **README.txt** - 사용 설명서 (이 파일)

## 🚀 빠른 시작

### 방법 1: AI 기능 사용 (추천)
1. **서버실행.bat** 더블클릭
2. 명령창이 열리면 브라우저에서 입력:
   ```
   http://localhost:8000
   ```
3. AI 자동 입력 기능 사용 가능!

※ Python이 설치되어 있어야 합니다.

### 방법 2: AI 없이 사용
1. **index.html** 파일을 더블클릭
2. 수동 입력으로 채용공고 생성
3. 모든 기능 정상 작동 (AI만 제외)

## 🤖 AI 기능 사용법

### Windows
```
서버실행.bat 더블클릭
→ http://localhost:8000 접속
```

### Mac / Linux
```bash
python3 server.py
→ http://localhost:8000 접속
```

## 📝 주요 기능

✅ 필수 항목 입력으로 자동 채용공고 생성
✅ AI 자동 입력 (텍스트 붙여넣기 → 마크다운 리스트로 자동 변환)
✅ 실시간 미리보기
✅ HTML 코드 복사
✅ 인쇄 기능
✅ HTML 파일 다운로드

## 💡 AI 자동 입력 팁

1. 기존 채용공고 텍스트 복사
2. "AI 자동 입력" 박스에 붙여넣기
3. "🤖 텍스트 자동 분석" 버튼 클릭
4. 자동으로 마크다운 리스트로 변환됨!

예시:
```
입력: 주요업무 - 인권 상담, 사건 처리, 교육 기획
출력: 
- 인권 상담 및 사건 처리
- 인권 교육 기획 및 실행
- 센터 운영 지원
```

## ⚠️ 문제 해결

### AI 기능 "Failed to fetch" 에러
→ **서버실행.bat** 실행 후 `http://localhost:8000` 접속

### Python이 없다고 나옴
→ Python 설치 필요 (https://python.org)
→ 또는 AI 기능 없이 수동 입력 사용

### 미리보기가 안 보임
→ 필수 항목(*)을 모두 입력했는지 확인

### 포트 8000이 사용 중
→ server.py 파일을 메모장으로 열기
→ PORT = 8000 을 PORT = 8080 으로 변경
→ 브라우저에서 http://localhost:8080 접속

## 🔧 기술 정보

- **CORS 문제 해결**: server.py가 프록시 역할
- **로컬 파일(file://)**: AI 기능 작동 안 함
- **웹서버(http://)**: AI 기능 정상 작동
